# 08 — Regras Editoriais

Este documento existe para proteger a identidade deste projeto.

À medida que a documentação crescer, existe o risco natural de transformar esta jornada em algo que ela nunca pretendeu ser.

Estas regras ajudam a manter o foco.

## Regra principal

Este repositório não existe para ensinar uma metodologia definitiva.

Ele existe para documentar uma jornada real.

As experiências registradas aqui representam o que funcionou, o que falhou, o que mudou e o que foi aprendido ao longo do caminho.

Nem toda prática descrita aqui será universal.

Nem toda ferramenta continuará sendo utilizada no futuro.

O objetivo não é prescrever uma única forma correta de criar software com IA.

O objetivo é registrar, de forma honesta e transparente, uma experiência real de aprendizado, construção e evolução.

Se este material ajudar outras pessoas a criar suas próprias soluções, então ele terá cumprido seu propósito.

## O que este projeto deve ser

- uma documentação viva;
- um diário de desenvolvimento;
- um registro de experimentos;
- uma coleção de aprendizados reais;
- um espaço para reflexões honestas;
- um registro da evolução pessoal e técnica.

## O que este projeto não deve se tornar

- um curso gigante;
- uma metodologia concorrente ao AISDD;
- uma plataforma complexa;
- uma coleção excessiva de arquivos e burocracia;
- um repositório de marketing;
- uma promessa de enriquecimento rápido;
- uma defesa de que IA substitui pessoas.

## Relação com o AISDD

```txt
AISDD
↓
O método.

software-creation-with-ai
↓
A história.
```

O AISDD documenta processo, estrutura e engenharia.

Este repositório documenta a jornada humana que levou até essas descobertas.

## Princípios editoriais

1. Honestidade acima de marketing.
2. Evidências acima de opiniões.
3. Experiência real acima de teoria abstrata.
4. Clareza acima de complexidade.
5. Reflexão acima de perfeição.
6. Humanos continuam responsáveis pelo resultado.

## Pergunta de validação

Antes de adicionar qualquer novo documento, perguntar:

```txt
Isto ajuda a contar a história?
```

Se a resposta for não, provavelmente este conteúdo pertence a outro repositório.
